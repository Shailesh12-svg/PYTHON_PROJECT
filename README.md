🛡️ Intrusion Detection Hybrid System (IDHS)

An advanced Hybrid Intrusion Detection System (HIDS + NIDS) powered by Machine Learning and Cybersecurity Analytics to detect malicious network activities, cyber threats, and abnormal system behavior in real-time.

📌 Project Overview

The Intrusion Detection Hybrid System combines both:

Signature-Based Detection → Detects known attacks using predefined patterns.
Anomaly-Based Detection → Uses Machine Learning models to identify unknown or suspicious activities.

This hybrid approach improves:

Detection Accuracy
Threat Intelligence
Real-Time Monitoring
False Positive Reduction

The system is designed for:

Network Security Monitoring
Threat Detection
Cyber Attack Analysis
Security Research
SOC (Security Operations Center) Simulation
🚀 Features
